据公告内容，獐子岛详细披露了公司2009年至2014年期间播苗及采捕海域情况， 包括各海域2014年以前的历年播苗时间、采捕时间及其对应关系。就媒体报道的獐子岛“收走” 约20艘航海日志的，公司称2014年海洋牧场灾后重建时，为强化管理，公司完善了制度体系， 加强了制度执行，而将航海日志收回统一归档就属于公司加强对船舶安全操作的规范化管理 。

就深交所“是否存在提前采捕于2013年底，止于2014年10月的情形”一问， 獐子岛回复称公司2013年11、12月至2014全年均按照采捕计划进行采捕，未组织进行“提前采捕”。 其中公告称公司重点核查了媒体报道提及的辽长渔15166—15170五条船只2013年11月至2014年12月的采捕区域， 公告同时披露相关船只实际采捕区域经纬度，表示上述五条船在该时间段均未涉及提前进入受灾区域采捕的情形。
